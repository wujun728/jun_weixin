#项目技术说明
1.项目采用最流行的基础框架:SpringMvc+MyBatis+Layui
2.本着不再发明轮子的思想，本项目缓存采用，红薯大大的J2Cache框架，具体参考此框架。
3.样式中，增加了scss，推荐使用IDEA编译

#项目组织架构
css   (项目样式)
font  (项目字体)
js    (项目脚本)
images(项目图标)
views (项目所有页面)
java后台中，core主要包含各种公共工具方法，
wxapi主要封装微信的功能，
wxcms主要包含后台框架的业务逻辑。

#功能模块
·运营数据：系统消息发送、粉丝数及访问情况。
·账号信息：公众号的详细配置信息。
·文本消息：关键字回复配置。
·图文管理：单图文和多图文的管理。
·菜单管理：自定义菜单的操作。
·粉丝管理：微信粉丝的详情及操作。
